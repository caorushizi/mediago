# m3u8 /bilibili 资源嗅探下载器 (MediaGo 助手)

本脚本由 **zhecydn** 提供技术支持与教程分享。通过此脚本，你可以直接在浏览器页面完成视频地址的提取与投喂，无需手动复制链接，极大提升了 MediaGo 的使用效率。

::: tip 外部教程参考
详细图文教程请访问：[zhecydn 的博客](https://blog.zhecydn.asia/archives/1962)
:::

## 🚀 快速开始

### 1. 安装环境
1. **安装插件**：在浏览器中下载并安装 **Tampermonkey** (油猴) 插件。
2. **电脑本地或者nas部署下载了mediago，电脑端不能关闭mediago**。

### 2. 添加脚本
点击下方链接进入 Greasy Fork 进行安装：
👉 [安装脚本：m3u8 资源嗅探器](https://greasyfork.org/zh-CN/scripts/559910)

## 📖 使用指南


![脚本界面](https://raw.githubusercontent.com/zhecydn/M3U8-MediaGo-/main/js.png) 


### 基础操作
1. **启用脚本**：在油猴插件中确认脚本已开启，然后打开想要下载视频的网页。
2. **唤出面板**：在视频播放页面会自动出现脚本的操作面板。
3. **界面交互（最上方）**：
   - **最小化**：点击面板最左侧按钮可折叠为齿轮图标，再次点击恢复。
   - **主题切换**：齿轮右侧图标可切换深浅色模式。
   - **配置地址**：点击**齿轮图标**可以填写你的 MediaGo 服务地址（下载到 NAS/Docker 用户必填）。

## 🛠️ 核心功能详解

### 1. 下载目标 (Target)
* **本地**：适用于下载视频到当前的电脑设备。
* **Docker**：适用于将视频投喂到远程 Docker 版本的 MediaGo。

### 2. 下载模式 (Mode)
* **API 模式**：**首选推荐**。实现静默一键下载，如若无法使用再选择url模式。
* **URL 模式**：备选方案。点击后会跳转到 MediaGo 页面进行手动下载。

### 3. 多选与批量投喂
* **多选/全选**：支持勾选多个视频文件进行批量操作。
* **一键投喂**：勾选后点击右上角“一键投喂”按钮，即可将选中的视频全部发送至 MediaGo。
* **温馨提示**：请勿一次性下载过多文件，以免造成mediago和脚本卡顿。

### 4. 任务归类
* **域名文件夹**：根据当前所在的域名自动创建目录归类下载（目前**仅在 Docker 版本**中有效）。
* **默认根目录**：视频将直接保存至 MediaGo 预设的根目录。
